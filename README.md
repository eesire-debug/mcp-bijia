# mcp-bijia

> 比价 MCP 服务器 — 中文电商全网最低价查询

**中文命名**，专为中国 AI Agent 生态设计。对接淘宝、京东、拼多多等平台，搜索+比价+购买链接一体化。

## 功能

- **全网比价** — 一次查询，返回多平台价格排名
- **最低价提醒** — 监控商品价格，降价通知
- **购买链接** — 附带返利链接（淘宝联盟/京东联盟）

## 安装

```bash
npm install -g mcp-bijia
```
